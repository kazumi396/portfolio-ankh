# Ankh of Eternity（架空ペルシャ絨毯ブランドサイト）

## 概要

HelloMentor から支給されたデザインを元に制作した、架空のペルシャ絨毯ブランドサイト「Ankh of Eternity」です。
Figma のデザインカンプをもとに、**レスポンシブ対応の静的コーディング**を行い、**アニメーションやスクロール演出などのインタラクション**を実装しました。
構造設計・コードの保守性・再利用性を重視し、実務を想定した制作を行っています。

## 制作期間

2025 年 1 月（約 1 週間）

## 使用技術

### フロントエンド

- HTML5 / CSS3
- Sass（SCSS 記法）
- バニラ JavaScript
- GSAP / ScrollTrigger（スクロール演出）

### JavaScript（動き・UI）

- ハンバーガーメニューの開閉
- スクロール連動によるヘッダー背景表示
- ファーストビューのオープニングアニメーション（初回訪問時のみ）
- スクロール連動のフェードイン・スタッガーアニメーション
- 背景テキストのスクロール連動横移動（matchMedia によるレスポンシブ対応）

## 工夫した点

- **レスポンシブ対応**：メディアクエリを活用し、デバイス幅に応じてレイアウト・フォントサイズ・余白を最適化しました。
- **コンポーネント化**：ヘッダー・ボタン・タイトルなどの共通パーツを部品化し、再利用性を高めています。
  JavaScript も機能ごとにモジュール化し、SCSS と合わせてコンポーネント単位で保守性を確保しました。
- **UI アニメーション**：GSAP + ScrollTrigger によるスクロール連動演出を実装。動きの自然さとパフォーマンスを両立させました。
- **オープニング演出の最適化**：sessionStorage を使用し、初回訪問時のみオープニングアニメーションを再生。リピーターには即座にコンテンツを表示します。
- **背景テキストの統一設計**：brand セクションと subpage セクションで共通の background-image コンポーネントを使用し、スクロール連動の横移動を実装しました。
